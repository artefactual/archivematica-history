During ingest, Archivematica optionally creates a Dissemination Information Package (DIP) for upload into an access system. Archivematica comes bundled with a default access system called ICA-AtoM, which supports hierarchical archival description and digital object display. The access tab links Archival Information Packages to the uploaded DIPs.
